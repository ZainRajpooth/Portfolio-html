To change webpage background color use Internal CSS by typing style background-color color-name /style
Whereas, External CSS is calling a cascaded file that is an aggregates of all the styles, that can be called upon for multiple HTML files

The above color has been changed using style i.e. Inline CSS

style color green

style border 1px solid red

style background-color yellow

style background-color rgb(255, 99, 71) i.e. RGB (Red,Green,Blue) with values 0-255

style background-color rgb(255, 99, 71, 0.3) i.e. RGBA (Red,Green,Blue,Alpha), where alpha being the transarency 0,0.1,0.2,...,0.9,1 i.e.(0-1)

style background-color rgb(200, 200, 200) i.e. if *values are same x,x,x = different shades of gray

style background-color #ff0000, for #rrggbb, i.e. red,green,blud with values 0-f

style background-color #b6b6b6 i.e. if *values are same x,x,x = different shades of gray

style background-color hsl(0,100%,50%), where hsl being hue, saturation, lightness with values 0-359,0-100%,0-100% respectively.

style background-color hsl(60,60%,60%) i.e. if *values are same x,x,x = color is not gray

style background-color hsla(70,70,70,0.8) where 'a' being alpha for transparency.
